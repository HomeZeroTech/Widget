<!--

Feenstra — Dropdown + 2 CTA's per maatregel (dual-flow)

Eén titel, dropdown (single-select) met ingebouwde iconen. Per maatregel een offerte-flow (CTA1) en een adviesgesprek-flow (CTA2) via data-tile-{key}-cta2-url. Maatregelen zonder cta2-url (hier: Zonnestroomboiler) tonen automatisch één knop. De AI-chat verschijnt als losse tekstlink onder de onderste knop zodra de ChatWidget aanwezig is.

Zonder ChatWidget blijft de tekstlink verborgen.
▶ Bekijk embed code 
<!--
  Dual-flow: per maatregel een offerte-flow (data-tile-{key}-url) + adviesgesprek-flow (data-tile-{key}-cta2-url).
  data-cta2-action="flow"   = HomeZero-leadflow (mét adresvelden + offline.html-fallback)
  data-cta2-action="booking"= externe agenda (Calendly/Google), opent direct zonder fallback
  Maatregel zonder cta2-url toont automatisch maar één knop.
-->
<hz-embed
  data-mode="scan"
  data-color="#e8762a"
  data-title="Waarin ben je geïnteresseerd?"
  data-tile-display="dropdown"
  data-tiles-max-select="1"

  data-tile-airco-title="Airco"
  data-tile-airco-url="https://...offerte-airco"
  data-tile-airco-cta2-url="https://...advies-airco"
  data-tile-solarboiler-title="Zonnestroomboiler"
  data-tile-solarboiler-url="https://...offerte-boiler"   <!-- geen cta2-url ⇒ 1 knop -->

  data-cta1-text="Direct een offerte"
  data-cta2-show="true"
  data-cta2-action="flow"
  data-cta2-text="Adviesgesprek aanvraag"

  data-address-format="dutch"
  data-show-email="true"

  data-ai-chat-show="true"
  data-ai-chat-text="Of chat met Fleur de Ai adviseur"
></hz-embed>

Scan — Multi-select tegels + combinatie-flows

Meerdere tegels mogelijk. 1 geselecteerd → de flow van díe tegel; meerdere geselecteerd → data-cta1-combo-url (1 combinatie-leadflow). De secundaire CTA gebruikt hier een globale generieke flow (data-cta2-url) die voor alle maatregelen herbruikt wordt.

▶ Bekijk embed code 
<hz-embed
  data-mode="scan"
  data-tile-display="large"        <!-- max-select weglaten = multi -->
  data-tile-zon-url="https://...zon"
  data-tile-batterij-url="https://...batterij"
  ...
  data-cta1-text="Bereken mijn besparing"
  data-cta1-combo-url="https://...combinatie"   <!-- bij >1 selectie -->
  data-cta2-show="true"
  data-cta2-action="flow"
  data-cta2-text="Plan een gratis adviesgesprek"
  data-cta2-url="https://...adviesgesprek"      <!-- globale generieke 2e flow -->
></hz-embed>

Scan — Custom iconen via base64 SVG attribuut

Elk tegel-icoon kan overschreven worden via data-tile-{key}-icon-svg met een base64-gecodeerde SVG. De ingebouwde iconen worden alleen als fallback gebruikt.

▶ Bekijk embed code 
<!--
  data-tile-{key}-icon-svg = base64-gecodeerde SVG string
  Genereer met: btoa(svgString) in JavaScript
  Of via Lovable generator: upload SVG of genereer met AI
-->
<hz-embed
  data-mode="scan"
  data-color="#0d9f6e"

  data-tile-solarpanels-url="https://..."
  data-tile-solarpanels-title="Zonnepanelen"
  data-tile-solarpanels-icon-svg="PHN2ZyB4bWxu..."

  data-tile-homebattery-url="https://..."
  data-tile-homebattery-title="Thuisbatterij"
  data-tile-homebattery-icon-svg="PHN2ZyB4bWxu..."

  data-tile-heatpump-url="https://..."
  data-tile-heatpump-title="Warmtepomp"
  data-tile-heatpump-icon-svg="PHN2ZyB4bWxu..."

  data-tile-carcharger-url="https://..."
  data-tile-carcharger-title="Laadpaal"
  <!-- carcharger gebruikt ingebouwd icoon als fallback -->

  data-show-email="true"
></hz-embed>

Scan — met AI Adviseur CTA (detectie)

De tweede knop verschijnt alleen als window.ChatWidget aanwezig is op de pagina. Gebruik de toggle hieronder om de ChatWidget te simuleren.

De knop verschijnt direct zodra je de simulatie aanzet — geen refresh nodig.
▶ Bekijk embed code 
<!-- ChatWidget moet ook op de pagina staan -->
<link rel="stylesheet" href="https://homezerotech.github.io/files/salesagent/widget.css">
<script src="https://homezerotech.github.io/files/salesagent/widget.js"></script>
<script>
  ChatWidget.init({ chatbotApiUrl: '...', apiKey: '...' });
</script>

<hz-embed
  data-mode="scan"
  data-color="#2A6DF4"
  data-cta2-show="true"
  data-cta2-text="of start een chat met onze AI adviseur"
  data-cta2-action="ai-chat"
  ...
></hz-embed>
