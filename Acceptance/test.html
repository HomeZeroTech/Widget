Scan — Tegels + adres + dual CTA

▶ Bekijk embed code 
<!--
  data-cta2-action (default: "pico") stuurt een direct-contact lead via de Pico API.
  Daarvoor is data-pico-key vereist. Zonder key werkt CTA2 niet.
  Alternatief: data-cta2-action="ai-chat" opent de AI chatwidget (geen Pico key nodig).
-->
<hz-embed
  data-mode="scan"
  data-color="#2A6DF4"
  data-gradient-from="#2A6DF4"
  data-gradient-to="#1a4fc4"
  data-tile-solarpanels-url="https://..."
  data-tile-solarpanels-title="Zonnepanelen"
  data-tile-homebattery-url="https://..."
  data-tile-homebattery-title="Thuisbatterij"
  data-tiles-default="solarpanels,homebattery"
  data-cta1-text-scan="Bereken wat je bespaart"
  data-cta1-text-booking="Plan een gratis adviesgesprek"
  data-cta2-show="true"
  data-cta2-text="Direct contact (30 sec)"
  data-cta2-action="pico"
  data-contact-skip-address="true"
  data-pico-key="your-api-key"
  data-pico-env="production"
  data-show-phone="true"
  data-show-email="true"
></hz-embed>

Scan

Scan — met AI Adviseur CTA (detectie)

De tweede knop verschijnt alleen als window.ChatWidget aanwezig is op de pagina. Gebruik de toggle hieronder om de ChatWidget te simuleren.

De knop verschijnt direct zodra je de simulatie aanzet — geen refresh nodig.
▶ Bekijk embed code 
<!-- ChatWidget moet ook op de pagina staan -->
<link rel="stylesheet" href="https://homezerotech.github.io/files/salesagent/widget.css">
<script src="https://homezerotech.github.io/files/salesagent/widget.js"></script>
<script>
  ChatWidget.init({ chatbotApiUrl: '...', apiKey: '...' });
</script>

<hz-embed
  data-mode="scan"
  data-color="#2A6DF4"
  data-cta2-show="true"
  data-cta2-text="of start een chat met onze AI adviseur"
  data-cta2-action="ai-chat"
  ...
></hz-embed>
